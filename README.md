# Siroco BGM Changer

Siroco BGM Changer 는 던전앤파이터 '무형의 시로코 레이드' 에서 '추적' 단계 '법칙의 관문' 에 등장하는 '이름을 잊은 수문장' 의 기믹중 하나인 순서를 미리 알 수 있게 '법칙의 관문' 에 입장하자 마자 나오는 BGM 을 다른 직관성 있는 BGM 으로 바꿔서 보다 알기 쉽게 만들어줍니다.
